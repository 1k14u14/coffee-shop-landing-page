# ☕ The Coffee Landing Page
A premium, responsive coffee shop landing page built with Vanilla HTML, CSS, and JavaScript.

## Features
- **Hero Section:** High-impact visual with smooth scrolling.
- **Dynamic Menu:** 30+ items across Coffee, Tea, and Food categories.
- **Photo Box Experience:** Integrated service description and customer gallery.
- **Responsive Design:** Optimized for mobile and desktop.
- **Scroll Animations:** Built using the Intersection Observer API.

## Tech Stack
- HTML5
- CSS3 (Custom Grid & Flexbox)
- Vanilla JavaScript
